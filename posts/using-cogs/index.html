In the last few years, rasters presented as Cloud-Optimized GeoTIFFs (COGs) have gained a lot of excitement in the field. A COG is a geoTIFF (.tif) hosted on a cloud or file server such as AWS S3, Microsoft Azure, or Apache, whose format is optimized for remote reads. While there’s an increasing body of knowledge and use cases, COGs are still shy of mainstream. In a recent project, we found it helpful to structure our raster data as COGs.
